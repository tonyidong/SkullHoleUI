This tool is made to draw curves on an input mesh.

How to use?
1. Open a mesh. 
2. Press Control or Button "Draw", then Press Left Mouse Button and move mouse to draw lines on the mesh.
3. Press Shift or Button "Select", then move the mouse close to the curve you want to delete and press left mouse button to delete.
4. Save curves to file. The curve filename is the same with the mesh's.
5. Open curve file to show curve network.
